获取并安装Intel MKL数值例程库

发表于 2018-05-27 | 更新于 2019-05-22 | 分类于 python | 评论数： | 阅读次数：

Intel Math Kernel Library（MKL）提供经过高度优化和大量线程化处理的数学例程,面向性能要求极高的科学、工程及金融等领域的应用。许多高性能数值计算包可以链接到MKL，例如Python的NumPy、Scipy，MATLAB（已经包含），Julia等。现在MKL已经面向全部使用者免费。本文介绍如何从Intel官网获取MKL库。

  1. 打开Intel Developer Zone，在右上角找到Sign up注册一个开发者账户。注册后网站会发送确认邮件。打开确认邮件中的确认链接完成注册（必要！！）。
  2. 打开Intel MKL网站，点击Free Download。
  3. 在页面右侧的表单中填写个人信息，点击Submit提交。
  4. 若之前已经登录，则提交后直接转向下载页面；若之前没有登录，点击右侧My Profile下面的Sign In登录。
  5. 下载对应的MKL版本。
