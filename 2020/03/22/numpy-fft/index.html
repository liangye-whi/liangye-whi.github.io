Numpy.fft模块求傅立叶级数

发表于 2020-03-22 | 更新于 2020-03-23 | 分类于 python | 评论数： | 阅读次数：

numpy.fft模块本来用于计算傅立叶变换，和傅立叶级数有关系但不是一回事。对于周期为$T$的实函数，如果我们能够获得其一个周期的函数值，实际上也是可以使用numpy.fft做傅立叶展开的。

叠加cos和sin函数的傅立叶展开
